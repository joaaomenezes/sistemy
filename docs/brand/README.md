# Identidade Visual e Design System Azzys

Esta pasta é a fundação oficial da identidade visual da Azzys.

Azzys é uma plataforma ERP/SaaS moderna com identidade Dark Tech Premium. O sistema visual usa fundos dark navy, superfícies azuladas profundas, verde/teal como cor principal da marca, azul como cor secundária tecnológica, Inter como fonte da interface e uma direção de logo geométrica e premium.

Use esta pasta antes de fazer qualquer alteração visual nas páginas. Ela define direção de marca, cores, tipografia, regras de logo, uso de ícones, fundações de componentes e o caminho recomendado de implementação.

## Arquivos

Documentação:

- `brand-guide.md` - estratégia da marca, personalidade, direção visual e o que evitar.
- `color-palette.md` - sistema oficial de cores e regras de uso.
- `typography.md` - tipografia do sistema, tipografia da logo e escala tipográfica.
- `logo-guidelines.md` - conceito da logo, versões, uso correto e uso incorreto.
- `favicon-checklist.md` - checklist de favicon, app icon e manifest.
- `icon-system.md` - biblioteca de ícones, estilo, tamanhos, cores e mapeamento por módulo.
- `ui-components.md` - regras visuais para botões, cards, inputs, modais e badges.
- `application-guidelines.md` - guia prático para sidebar, topbar, landing, login, PDV e financeiro.
- `implementation-plan.md` - plano de implementação por fases.

Fundações CSS:

- `tokens.css` - tokens oficiais do design system.
- `design-system.css` - classes base reutilizáveis criadas a partir dos tokens.

Esta pasta não redesenha o produto sozinha. Ela é a camada de referência para trabalhos futuros de implementação visual.
