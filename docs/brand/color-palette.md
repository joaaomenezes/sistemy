## Cor Principal da Marca

`#00D084`

Usada para:

- Identidade principal da marca
- CTA principal
- Sucesso
- Valores positivos
- Confirmações
- Destaques primários

## Cor Principal em Público/PDV

`#00C896`

Usada para:

- Landing page
- Autenticação
- PDV
- CTAs públicos

## Cor Secundária Tecnológica

`#2F80FF`

Usada para:

- Informações
- Gráficos
- Links
- Ícones
- Destaques secundários
- Visualização de dados

## Azul Secundário Público

`#0077FF`

Usado para:

- Destaques em páginas públicas
- Gradientes
- Visuais de produto
- Ênfase informativa

## Gradiente Oficial da Marca

```css
--gradient-brand: linear-gradient(135deg, #00D084 0%, #00C896 45%, #2F80FF 100%);
```

## Regras de Uso

- Não use verde para tudo.
- Não use azul como única cor de marca.
- Não exagere no uso de gradientes.
- Use vermelho apenas para erro, perigo, vencido, cancelamento ou status negativo.
- Use laranja/amarelo para atenção e alertas.
- Use roxo para IA, recursos premium ou insights avançados.
- Mantenha as superfícies escuras consistentes.
- Evite preto puro, exceto em áreas muito pequenas e intencionais.
- Mantenha textos secundários legíveis.
