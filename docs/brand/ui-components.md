# Componentes de UI

Estas são fundações visuais, não uma biblioteca completa de componentes.

## Botões

Botão principal:

```css
background: linear-gradient(135deg, #00D084 0%, #00C896 100%);
color: #021C14;
border-radius: 12px;
font-weight: 700;
```

Botão secundário:

```css
background: transparent;
border: 1px solid rgba(255,255,255,.09);
color: #E8EDF5;
border-radius: 12px;
```

Botão informativo:

```css
background: #2F80FF;
color: #FFFFFF;
```

Botão de perigo:

```css
background: #FF4D6D;
color: #FFFFFF;
```

## Badges

Use fundos suaves com texto legível.

Significados recomendados:

```text
Pago: verde
Recebido: verde
Pendente: âmbar
Vencido: vermelho
Cancelado: vermelho
Ativo: verde
Inativo: muted
Aviso: laranja
Info: azul
```
