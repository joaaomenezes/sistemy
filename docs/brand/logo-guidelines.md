# Diretrizes da Logo

## Conceito da Logo

A logo da Azzys deve ser baseada em uma letra "A" geométrica.

O símbolo deve representar:

- Estrutura
- Crescimento
- Gestão empresarial
- Tecnologia
- Direção
- Automação
- Inteligência
- Conexão

O "A" pode conter um detalhe sutil em espaço negativo sugerindo:

- Fluxo
- Caminho de decisão
- Conexão
- Um "Y" oculto

O "Y" oculto é opcional e não deve deixar o símbolo complexo.

## Estilo da Logo

A logo deve ser:

- Minimalista
- Geométrica
- Premium
- Limpa
- Escalável
- Reconhecível em tamanhos pequenos
- Adequada para uma plataforma SaaS/ERP

## Logo Principal

```text
[Símbolo A geométrico] AZZYS
```

## Cores da Logo

Versão sobre fundo escuro:

```text
Símbolo: gradiente #00D084 -> #00C896 -> #2F80FF
Wordmark: #FFFFFF
```

Versão sobre fundo claro:

```text
Símbolo: gradiente #00D084 -> #00C896 -> #2F80FF
Wordmark: #050816 ou #0B1220
```

Versões monocromáticas:

```text
Branco
Dark navy
Verde principal
```

## Versões Necessárias da Logo

```text
logo-horizontal-dark.svg
logo-horizontal-light.svg
logo-symbol.svg
logo-symbol-dark.svg
logo-symbol-light.svg
logo-monochrome-white.svg
logo-monochrome-dark.svg
```

## Uso

Use a logo horizontal em:

- Landing page
- Login
- Header
- Apresentações
- Documentos institucionais

Use o símbolo isolado em:

- Sidebar recolhida
- Favicon
- App icon
- PWA
- Loader
- Espaços pequenos

## Uso Incorreto

Não:

- Distorcer a logo
- Rotacionar a logo
- Alterar as cores aleatoriamente
- Adicionar sombras pesadas
- Adicionar efeitos 3D
- Usar sobre fundo de baixo contraste
- Usar engrenagem como símbolo principal
- Usar gráficos genéricos como símbolo principal
- Usar visual cartoon ou gamer
