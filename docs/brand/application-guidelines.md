# Diretrizes de Aplicação

## Sidebar

- Use fundo dark navy.
- Use a logo Azzys no topo.
- Use somente o símbolo quando a sidebar estiver recolhida.
- Use ícones outline.
- O menu ativo pode usar fundo verde sutil ou gradiente.
- Evite brilho excessivo.

## Topbar

- Use superfície escura.
- O campo de busca deve ser limpo e sutil.
- Ícones de notificação, ajuda e perfil devem ser consistentes.
- Evite poluição visual.

## Landing Page

- Use fundo dark navy.
- Use headline grande em branco.
- Use gradiente da marca em uma palavra importante.
- Use CTA verde.
- Use cards escuros e bordas sutis.
- Mantenha o visual premium e direto.

## Login

- Use card escuro centralizado.
- Use a logo Azzys.
- Use CTA verde.
- Use texto simples.
- Evite excesso de informação.

## PDV

- Priorize clareza operacional.
- Use CTA forte para finalizar venda.
- Use botões claros para métodos de pagamento.
- Use cards de produto consistentes.
- Evite ruído visual decorativo.

## Financeiro / DRE

- Priorize legibilidade.
- Use vermelho apenas para despesas, valores negativos, vencidos e cancelamentos.
- Use verde apenas para valores positivos e recebidos.
- Use azul para informação.
- Não deixe todos os números coloridos.
