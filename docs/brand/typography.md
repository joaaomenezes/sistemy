# Tipografia

## Fonte do Sistema

Use Inter na interface do produto.

```css
font-family: 'Inter', system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

Use Inter em:

- Dashboard
- PDV / POS
- Financeiro
- DRE
- Clientes
- Produtos
- Estoque
- Relatórios
- Modais
- Tabelas
- Formulários
- Sidebar
- Topbar

## Fonte da Logo

Recomendação:

```text
Sora Bold
```

Alternativa:

```text
Space Grotesk SemiBold
```

## Regras da Wordmark

```css
font-family: 'Sora', sans-serif;
font-weight: 700;
letter-spacing: 0.08em;
text-transform: uppercase;
```

## Escala Tipográfica

```css
--font-xs: 12px;
--font-sm: 13px;
--font-base: 14px;
--font-md: 16px;
--font-lg: 18px;
--font-xl: 24px;
--font-2xl: 28px;
--font-3xl: 32px;
```

## Uso

```text
Título de página: 24px / 700
Título de seção: 18px / 700
Título de card: 15px ou 16px / 600
Texto normal: 14px / 400
Texto de tabela: 13px ou 14px / 400
Texto de botão: 14px / 600
Valor de KPI: 28px ou 32px / 700
Badge: 12px / 600
```

## Regras

- Não misture fontes em telas operacionais.
- Não use fontes futuristas ou gamer dentro da interface do ERP.
- A logo pode ter uma fonte própria; a interface do sistema deve usar Inter.
- Evite textos abaixo de 12px.
- Evite parágrafos em caixa alta.
- Mantenha valores financeiros com alta legibilidade.
