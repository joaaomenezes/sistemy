# Sistema de Ícones

## Biblioteca Recomendada

Se o projeto já usa Bootstrap Icons, mantenha Bootstrap Icons.

Recomendado:

```text
Bootstrap Icons
```

Alternativa:

```text
Lucide
```

Não misture múltiplas bibliotecas de ícones, exceto quando for absolutamente necessário.

## Estilo

- Estilo outline
- Linhas limpas
- Cantos arredondados quando possível
- Peso visual consistente
- Sem ícones 3D na interface operacional
- Sem emojis aleatórios na UI principal
- Sem mistura de ícones filled/outline sem propósito

## Tamanhos

```text
Ícones da sidebar: 18px a 20px
Ícones em botões: 16px
Ícones em cards/KPIs: 20px a 24px
Ícones de empty state: 40px a 64px
```

## Cores

```css
--icon-default: #94A3B8;
--icon-active: #FFFFFF;
--icon-primary: #00D084;
--icon-info: #2F80FF;
--icon-warning: #FF8C42;
--icon-danger: #FF4D6D;
--icon-purple: #A78BFA;
```

## Mapeamento Sugerido de Ícones

```text
Dashboard: ícone de grid/dashboard
PDV: ícone de carrinho/recibo
Vendas: ícone de sacola/carrinho
Financeiro: ícone de moeda/carteira
Clientes: ícone de usuários
Produtos: ícone de caixa
Estoque: ícone de arquivo/pacote
Relatórios: ícone de gráfico de barras
Configurações: ícone de engrenagem
Notificações: ícone de sino
Ajuda: ícone de pergunta
Sair: ícone de saída
Busca: ícone de busca
```
